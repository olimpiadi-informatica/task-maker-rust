Expand description

Adds methods for failing without panic. Like expect but without panic.

Required Methods§

Source

fn nice_unwrap(self) -> T

Fail exiting with 1 if the value is not present. Otherwise return the content.

Source

fn nice_expect<S: Display + Send + Sync + 'static>(self, mex: S) -> T

Fail exiting with 1 if the value is not present, printing to stderr the message. Otherwise return the content.

Source

fn nice_expect_with<S: Display + Send + Sync + 'static, F: FnOnce() -> S>( self, f: F, ) -> T

Fail exiting with 1 if the value is not present, printing the message returned by the provided function. Otherwise return the content.

Dyn Compatibility§

This trait is not dyn compatible.

In older versions of Rust, dyn compatibility was called "object safety", so this trait is not object safe.
